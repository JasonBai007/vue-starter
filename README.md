## How to use
### Install dependencies
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Preview your dist files
```
cd dist
http-server
```

### Stop your local http server
```
Ctrl + C
```
